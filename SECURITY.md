# Security Policy

## Supported Versions

Security fixes are applied to the default branch.

## Reporting a Vulnerability

Please do **not** open public issues for suspected vulnerabilities.

Report privately via GitHub Security Advisories:
- `https://github.com/matthiasSchedel/tiny-cli-tools-collection/security/advisories/new`

Include:
- affected package(s)
- reproduction steps
- impact assessment
- suggested remediation (if available)

You will receive an acknowledgment as quickly as possible, and we will coordinate disclosure after a fix is available.
